Add possibility to remove a tab via a button, close tab, and reopen them Also disable a specific tab and reable it.

Home

Usage of activateTabIndex and changeTab

See method addTabByActiveTabIndex and addTabByChangeTab

Goal 1 - Add tab

Add a new tab by using an array in the component and by activating the newly created tab. When using changeTab, it doesn't work but by ativeTabIndex it works

Try by clicking on "Add tab (activateTabIndex)" and next by clicking on "Add tab (changeTab)"

Goal 2: Set isDisabled and switch to a specific tab

Lock a tab (isDisabled=false) and switch to the first tab. By using changeTab it works and by using activeTabIndex it doesn't.

Scrollable tab content hide euiTabsRightContent

When you add tabs and the scrollable content appears, it hides the euiTabsRightContent.

Try by clicking multiple times on "Add tab (click until hiding this right content)"

Activate tab outside of the scrollable view is not showing the tab (only content)

When you have a lot of tabs, the scrolling is activated. When the last tab is selected, the first one can be outside of the view (on the left). If a tab outside of the scrollable view is selected programmatically, the content is shown but the tab is not show.

To reproduce, click multliple time on "Add tab (Activate index)" until the Home tab is hidden and select the last tab. Click on"Delete tab". Content from Home will be showned but not the tab label.
