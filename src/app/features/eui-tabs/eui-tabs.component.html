Add possibility to remove a tab via a button, close tab, and reopen them Also disable a specific tab and reable it.

Home

In this use case, you have 2 hardcoded tabs that the user will not be able to delete or close.
The user will have the possibility to add tabs that we will called UserTabs. The user is allowed to closed the tab by using the "x" on the tab itself og by using a button in the content. The user will be allow to reopen all the tabs by clicking on "Recover tabs".

It is possible to "Add tab" which will call a method in the component.ts file to add the element in the corresponding array. Once done, the tab is activated by getting the index and setting the activeIndex.
